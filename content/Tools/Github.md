Github is a site to hold a version control line for your projects, which pretty much means that it helps back up whatever progress you make on your projects. It also lets you host those projects online, and you can even let people collaborate on those projects too!

It also has a nice webpage hosting feature, which I am using to keep this site up! Thanks Github!

This one is pretty essential, you should probably learn how to use Git in order to make the most of what Github has to offer.
