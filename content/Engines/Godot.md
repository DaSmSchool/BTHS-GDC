Godot is an open source 2D/3D game engine that is *very* similar to other modern game engines like Unity and Unreal Engine. It can be used to make pretty graphically powerful games, and has a large and quickly growing community of members. In my opinion, it's much easier to approach than Unity or Unreal Engine, though you *will* need some time/advice to orient yourself around the engine, since it will feel very overwhelming the first time around. It primarily uses its own native language, GDScript, though C# is supported.

If you have not tried to make a small project with any other engine, I would not jump to Godot just yet, but it's definitely *very* fun to use if you start climbing its learning curve. The process of making larger projects is much more flexible here, and you could very well end up making some strong games with Godot!

Godot's editor runs on Windows/Mac/Linux/Android/HTML.
