If you don't necessarily want to start making games by writing them from the ground up, you can also try out modding games to start off with a pretty large framework that works! If you want to expand upon a game that you already really like, modding lets you do exactly that! Some game modding communities have even made up ways to mod games without writing any code by yourself! You can even find out ways to mod games that are on consoles! Awesome!

Unfortunately, there is no universal source for all game mods, but if you google "how to mod ---" and you end up with some solid results, you might be able to find a community of modders for that game if you dig around a little (provided that game's community is large enough for a live modding scene, or just somehow has a modding scene larger than its official communities. It depends!) From there, you can ask them how to mod the game directly, and they can give you the tools necessary to mod that game.
