Love2D is a 2D (who would have guessed!) game framework that uses the Lua programming language. It mostly relies on you programming most of the game functions outside of basic drawing/sound/grunt work that a game framework will normally do for you. It's fairly powerful, and it has been used to make a whole bunch of indie games that came out recently, like [Balatro](https://www.playbalatro.com)!

If you don't like the restrictions that [[PICO-8]] has, but you still want to make a game with a lightweight engine in Lua, try this out!
You should only avoid this if you want to do anything in 3D. It's a great choice otherwise!

Love2D is available on Windows/Mac/Linux/Android/iOS, though the mobile versions are pretty dated.
