Scratch is a block based programming language/engine that is geared towards those who have little to no prior coding experience, and has a pretty large community of users who publicly post and remix projects on Scratch's website. It's fairly easy to pick up for the first time!

If you don't know where to start with programming at all, check this one out!

If you have tried Scratch already, I would suggest you try to learn some other engine that I listed here.

It's available on Web Browsers, Windows, Mac and Android.
