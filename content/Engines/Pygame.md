Pygame is a Python based graphics engine. Its site has a repository of game releases produced by its users, and is capable of 2D graphics. Its based on object oriented designs, and it helps if you know Python from something like APCSP.

If you know Python, and want to make a banger APCSP (Extended) project, try it out!

If you don't know very much about programming Classes and Objects/Object oriented programming, this might be annoying to use.

Runs on any device that can run Python version 3.x through a command line. This includes Windows, Mac and Linux. Technically possible on other platforms if you are hacky.
